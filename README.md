# 🚀 겨울 프로젝트 (Hero)
> 20225531 김태우의 자바 프로젝트입니다.

### 🎬 DEMO
- 현재 프로젝트는 로컬 환경에서 실행 가능하며, 추후 구동 영상을 첨부할 예정입니다.

### ✨ Features (주요 기능)
- 자바(Java) 기반의 핵심 기능 구현
- 객체 지향 설계를 통한 모듈화

### 🏗️ Architecture (구조)
- `Book.java` / `PrintablePanel.java` 등 컴포넌트 기반 구조

### 🏃 Runbook (실행 방법)
1. 프로젝트를 다운로드(Clone)합니다.
2. 메인 클래스를 실행합니다.

### 🛠️ Troubleshooting (문제 해결)
- 실행 중 인코딩 에러 발생 시, IDE의 UTF-8 설정을 확인하세요.
